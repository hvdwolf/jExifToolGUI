Installing jExifToolGUI

jExifToolGUI actually does not need to be installed. The only exception is the debian .deb file. All other release files can be copied into "some" folder and started from there.
The releases mentioning "including java V11" do not need anything else. The releases without the jre, require you to download either the full java V 11 (or newer) JDK or the V11 (or newer) JRE from https://adoptopenjdk.net/releases.html.

jExifToolGUI needs exiftool from Phil Harvey, but exiftool is not bundled with jExifToolGUI. You have to download it yourself from exiftool.org.

Linux Platforms

On Linux you have a few options.
  • On a debian based Linux (Debian/Ubuntu/Mint/MX Linux/Raspbian etc. etc.): You can download the jexiftoolgui-{some version}.deb.
    You can install this on any architecture that supports Java V11. Install it with sudo dpkg -i jexiftoolgui-{some version}.deb. It will install jExifToolGUI, add icons/shortcuts to your menu and install java V11 and "image-exiftool" for you.
  • The AppImage: This is a "universal" appimage (see https://appimage.org). It runs on any 64bit Intel Linux. (32bit and ARM architectures are currently not supported. In case of demand they can easily be made)

Windows

For Windows you have a few options, but the most simple one is to download the "jExifToolGUI-{some version}-win-x86_64.zip".
This is the "all in one" version and comes with the java V11 JRE (java runtime environment) included.
It does require a 64bit Windows.
Unzip this zip file into some folder. Make sure to unzip with paths intact! Simply double-click the .exe file to start jExifToolGUI.
With regard to exiftool: If you download the stand-alone executable, make sure to rename "exiftool(-k).exe" to "exiftool.exe" after unzipping the zip file.

MacOS

For MacOS there is the full bundle inside jExifToolGUI-x86_64-macos-full-{some version}.dmg.zip. It contains jExifToolGUI and also "inside" the bundle the java V11 JRE.
Download the zip and unzip it. Open the dmg as you are used. Drag the jExifToolGUI.app into the "Applications" or any place of your liking.

Universal

The jExifToolGUI.jar. The barebone jExifToolGUI.jar containing all dependencies.
This one is for those user who know what they are doing and only need the jar. You need to have java V11 installed on your system.
Start with java -jar jExifToolGUI.jar &
Or alternatively java -Xmx1024M -jar jExifToolGUI.jar &.
